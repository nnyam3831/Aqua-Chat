# Aqua-Chat
Real Time 채팅 어플리케이션 Aqua Chat입니다. 

* https://github.com/nnyam3831/aquachat-front
* https://github.com/nnyam3831/aquachat_server

## Project Stack
------------------
## Client
* React
* React-Router-DOM
* TypeScript
* Styled-Components
* Apollo

## Server
* Node.js
* TypeScript
* GraphQL
* TypeORM
* Type-GraphQL
